# Code Of Conduct

Contributors are expected to collaborate respectfully and professionally.

## Expected Behavior

- Be factual, constructive, and respectful.
- Prioritize safety, security, and user trust.
- Raise risks early with clear technical evidence.

## Unacceptable Behavior

- Harassment or discrimination.
- Intentional misuse of privileged access or secret material.
- Unsafe automation practices that bypass guardrails.

## Enforcement

Report concerns through maintainers using the process in `/Users/sdevisch/dev/orxaq-ops/SECURITY.md`.
